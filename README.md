# AI Chatbot using Python and NLTK

## Project Description
This project is a rule-based AI Chatbot developed as part of an internship assignment.  
The chatbot uses basic Natural Language Processing (NLP) techniques to interact with users via the command line.

---

## Objectives
- Implement a conversational chatbot using Python
- Apply basic NLP concepts using the NLTK library
- Demonstrate clean and modular coding practices

---

## Technologies Used
- Python
- NLTK
- Functions and modular programming
- Control flow statements

---

## NLP Techniques Applied
- Text normalization
- Tokenization
- Keyword-based intent recognition
